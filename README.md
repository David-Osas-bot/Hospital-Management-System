# Hospital Management System

A modern and responsive Hospital Management System built with **React**, **Tailwind CSS**, and **Firebase** to streamline hospital operations and improve healthcare service management.

This project was collaboratively developed by a team of NIIT students as part of a learning and practical software development experience, focusing on building scalable web applications and applying modern frontend technologies.

## Features

- Patient registration and management
- Doctor and staff information management
- Appointment scheduling system
- Secure authentication and user access control
- Real-time database integration with Firebase
- Responsive and user-friendly interface
- Dashboard for monitoring hospital activities
- Data storage and cloud services powered by Firebase

## Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Backend & Database:** Firebase
- **Authentication:** Firebase Authentication
- **Hosting:** Firebase Hosting (if applicable)

## Project Goal

The goal of this system is to digitize and simplify hospital administrative processes, reduce manual record handling, and provide an efficient platform for managing healthcare-related operations.

## Collaboration

This project was built collaboratively with fellow **NIIT students**, promoting teamwork, problem-solving, and hands-on experience in modern web development practices.

## Installation

```bash
git clone <repository-url>

cd hospital-management-system

npm install

npm start
```
